# Таймер обратного отсчета с динамическим склонением (Python)

### 📝 Задача
Реализовать программу для расчета точного времени, оставшегося до выхода игры (8 ноября 2022 года). 

Основные требования:
* Автоматический расчет разницы между введенной датой и датой релиза.
* Корректное склонение существительных (день/дня/дней, час/часа/часов) в зависимости от числительных.
* Вывод только значимых единиц времени (скрытие нулевых значений).
* Дружелюбный интерфейс и обработка ошибок ввода.

### 💡 Решение и производительность
Задача решена с использованием стандартной библиотеки `datetime` и кастомного алгоритма морфологического анализа.

**Особенности реализации:**
* **Точность:** Разница во времени вычисляется с точностью до минуты.
* **Грамматика:** Реализована логика выбора правильного окончания слова, что делает вывод программы естественным и профессиональным.
* **Надежность:** Код защищен от некорректного ввода даты и учитывает пограничные случаи (например, когда до релиза осталось меньше минуты).
* **Производительность:** Алгоритм работает за константное время **O(1)**, не перегружая систему строковыми преобразованиями, что гарантирует мгновенный отклик.

### 🛠 Стек
* **Язык:** Python 3.9+
* **Библиотеки:** `datetime` (работа с временными интервалами), `typing` (аннотации типов для стабильности кода).
