# Оптимизация конвертера данных EAV в Wide CSV (Python)

### 📝 Задача
Реализовать функцию `condense_csv()` для преобразования данных из формата **Entity-Attribute-Value (EAV)** (где каждая строка содержит имя объекта, его свойство и значение) в классический табличный вид (**Wide format**), сгруппировав свойства по идентификатору объекта.

### 💡 Решение и производительность
Изначальный алгоритм опирался на загрузку всего файла в оперативную память и использование вложенных циклов с математическим вычислением индексов. Это давало сложность **O(n * m)** по времени и **O(n)** по памяти, что делало код хрупким и непригодным для больших объемов данных.

Алгоритм был осознанно перепроектирован в пользу **потоковой обработки (Streaming Data)** с использованием архитектуры конечного автомата (State Machine) и хеш-таблиц.

Это обеспечивает:
* **Сложность по времени O(n):** Двухпроходное чтение с использованием словаря в качестве `OrderedSet` обеспечивает мгновенный поиск уникальных колонок за **O(1)** вместо линейного перебора списков.
* **Константное потребление памяти O(m) ≈ O(1):** Благодаря потоковой записи (`row_buffer`), в оперативной памяти в любой момент времени находится только один обрабатываемый объект. Это полностью исключает риск падения с ошибкой `Out of Memory` даже на файлах размером в сотни гигабайт.
* **Устойчивость и масштабируемость:** Алгоритм не зависит от количества свойств у объекта и гарантированно сохраняет изначальный порядок колонок (используя свойства `dict` в Python 3.7+).

### 🛠 Стек
* Python (модуль `csv`, `typing`)
* Концепции: Алгоритмическая сложность (Big O), Streaming Data Processing, State Machine, Type Hinting, Data Pivoting.

### 🚀 Как использовать

```python
from your_module import condense_csv

# Допустим, у нас есть файл 'data.csv' в формате EAV:
# ball,color,purple
# ball,size,4
# cup,color,blue
# cup,size,1

# Функция выполнит преобразование и создаст файл 'condensed.csv'
# Первый столбец получит пользовательское имя, например 'Object_ID'
condense_csv("data.csv", id_name="Object_ID")

# Результат в 'condensed.csv':
# Object_ID,color,size
# ball,purple,4
# cup,blue,1
```
