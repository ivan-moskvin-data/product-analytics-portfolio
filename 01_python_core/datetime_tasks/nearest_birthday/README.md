# Поиск ближайших дней рождения (Оптимизация потоковой обработки)

### 📝 Задача
Реализовать скрипт для поиска сотрудников, чей день рождения наступает в течение ближайших 7 дней от заданной даты. Если таких сотрудников несколько, необходимо вывести имена самых младших по возрасту (тех, чья дата рождения ближе всего к текущему дню). Алгоритм должен корректно обрабатывать високосные года (перенос 29 февраля на 1 марта в невисокосные годы).

### 💡 Решение и производительность
Скрипт спроектирован с упором на минимальное потребление ресурсов. Реализована потоковая обработка данных в один проход (O(n) по времени и O(1) по памяти). 

Это обеспечивает:
* **Экономию оперативной памяти (Memory Efficiency):** Использование `sys.stdin.readline()` вместо загрузки всего списка в память (`readlines()`) позволяет обрабатывать файлы бесконечного размера без риска `MemoryError`.
* **Высокую скорость:** Проверка 7-дневного окна и фильтрация самых младших именинников происходят "на лету" прямо во время чтения строки.
* **Надежную обработку Edge Cases:** Расчет високосных годов выполняется через встроенный механизм `replace` с перехватом `ValueError`, что дает алгоритмическую сложность вычисления следующего дня рождения O(1) без тяжелых итераций по календарю.

### 🛠 Стек
* Python (`datetime`, `sys`)
* Концепции: Потоковая обработка, Алгоритмическая сложность, Edge Case (Високосные года).

### 🚀 Как использовать
Скрипт читает данные из стандартного потока ввода (`stdin`)

**Формат входных данных (например, `input.txt`):**
```text
14.05.2024
3
Иван Иванов 16.05.1995
Анна Смирнова 19.05.1998
Петр Петров 19.05.1998
