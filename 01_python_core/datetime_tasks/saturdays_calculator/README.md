📝 Задача

Реализовать надежную функцию для точного подсчета количества суббот между двумя датами включительно.

💡 Решение и производительность

Изначально алгоритм был спроектирован через итеративный перебор дат (сложность O(n)), но в финальной версии полностью оптимизирован до математического расчета O(1) за счет использования модульной арифметики и целочисленного деления.

Это обеспечивает:

Мгновенный результат (Time O(1)) на любых временных интервалах — скорость вычислений не зависит от ширины диапазона (будь то 5 дней или 500 лет).

Отсутствие вычислительной деградации и экономию памяти (Space O(1)): мы не создаем в памяти списки порядковых номеров и избегаем лишних аллокаций.

Повышенную отказоустойчивость: встроенная защита от неверных типов данных и автоматическая нормализация обратного порядка дат.Идеально подходит для интеграции в высоконагруженные аналитические пайплайны и финтех-сервисы, где важна скорость работы с календарными графиками.

🛠 Стек

Python 3+ (datetime)

Концепции: Алгоритмическая сложность (O(1) против O(n)), Type Hinting, Defensive Programming.


🚀 Как использовать

```python
from datetime import date
from your_module import count_saturdays_between_dates

# Пример 1: Обычный порядок дат
start = date(2023, 10, 1)  # Воскресенье
end = date(2023, 10, 31)   # Вторник
saturdays = count_saturdays_between_dates(start, end)
print(f"Суббот: {saturdays}")  # Вывод: Суббот: 4

# Пример 2: Обратный порядок (функция автоматически отсортирует)
date1 = date(2025, 1, 1)
date2 = date(2024, 1, 1)
print(count_saturdays_between_dates(date1, date2))  # Вывод: 52
```
